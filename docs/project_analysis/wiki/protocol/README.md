## ⚠️ 安全模型

### 连接模式

#### ADB 模式 (推荐)

```
┌─────────┐      USB 线       ┌─────────┐
│  PC     │ ══════════════════│ 手机    │
│         │                   │         │
└─────────┘                   └─────────┘

安全性：
✓ 数据完全不经过网络
✓ ADB 自身提供加密
✓ 适用于所有环境（包括公司网络）
```

#### 网络模式 (受限)

```
┌─────────┐     UDP/TCP      ┌─────────┐
│  PC     │ ──────────────────│ 手机    │
│         │                   │         │
└─────────┘                   └─────────┘

安全性：
✗ 明文传输，可被监听
✗ 无认证，可被伪造
⚠ 仅限可信网络使用
```

### 安全对比

| 特性 | ADB + USB | ADB + tcpip | 网络模式 |
|------|-----------|-------------|----------|
| 数据经过网络 | 否 | 是 | 是 |
| 加密 | ✓ ADB | ⚠️ 弱 | ✗ 明文 |
| 认证 | ✓ | ✓ | ✗ |
| 适用场景 | 所有 | 可信网络 | 内网/隔离 |
| 推荐度 | ⭐⭐⭐ | ⭐⭐ | ⭐ |

### 安全建议

```
1. 生产环境：使用 ADB + USB 直连
2. 内网测试：网络模式可接受
3. 公网环境：禁止使用网络模式（除非 VPN）
4. 公司网络：必须使用 ADB + USB
```

## 核心常量速查

### UDP Header (24字节)

```
偏移  大小  字段            说明
────────────────────────────────────
0     4     sequence        包序号 (BE)
4     8     timestamp       时间戳 (BE)
12    4     flags           标志位 (BE)
16    8     send_time_ns    发送时间纳秒 (BE)
────────────────────────────────────
```
